# 🎯 The Perfect Guess Game

The Perfect Guess Game is a simple Python-based console game where the computer generates a random number and the player tries to guess it.

## 🚀 Features
- Random number generation (1–100)
- Interactive user input
- Smart hints (higher/lower)
- Tracks number of attempts
- Input validation (handles invalid entries)
- Replay option

## 🧠 How It Works
- The program generates a random number between 1 and 100.
- The player enters guesses via the console.
- The game provides feedback:
  - Too high → try lower
  - Too low → try higher
- The game ends when the correct number is guessed.
- Total attempts are displayed.

## 🛠️ Technologies Used
- Python
- Random module
